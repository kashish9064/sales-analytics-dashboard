# Sales Analytics Dashboard

## Overview

This project analyzes multi-year sales data using Excel, MySQL, and Power BI to identify revenue trends, product performance, seasonal patterns, and business KPIs.

The project follows an end-to-end analytics workflow including data preparation, SQL analysis, dashboarding, and business insight generation.

---

## Tools Used

- Excel
- MySQL
- Power BI
- DAX
- Pivot Tables

---

## Business Questions

- Which products generated the highest revenue?
- Which years experienced the strongest growth?
- What seasonal sales patterns exist?
- Which periods underperformed relative to yearly averages?
- How did revenue trends evolve over time?

---

## SQL Concepts Used

- JOINS
- SELF JOINS
- Subqueries
- CASE Statements
- Aggregations
- Indexing

---

## Dashboard Features

- KPI Cards
- Revenue Trend Analysis
- Product-wise Revenue Analysis
- Seasonal Analysis
- Interactive Filters

---

## Key Insights

- Product-level revenue contribution varied significantly across years.
- Revenue trends highlighted seasonal demand fluctuations.
- Certain years showed declining growth despite stable transaction volume.
- KPI analysis improved visibility into operational performance trends.
